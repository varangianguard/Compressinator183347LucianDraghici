There are two versions of this program (GUI and Console)

Console:
Type zip help for all the necessary info;

GUI:
Here you have to type in the directory path for any of the buttons you want to press.
EXAMPLE:
Path to Zip folder= C:\Users\Raf\Desktop (has to exist)
name= ziptest(can exist)
Uncompressed folder = C:\Users\Raf\Desktop\Test (has to exist)
Extract folder =  C:\Users\Raf\Desktop\ziptestExtracted (can exist)
Extension = .txt (specifies an extension for extraction or compression)
